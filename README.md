# FastAPI Social Media API

## Features

- User Registration
- JWT Authentication
- Posts CRUD
- Voting System
- PostgreSQL Database
- Alembic Migrations

## Tech Stack

- FastAPI
- SQLAlchemy
- PostgreSQL
- Alembic
- Pydantic

## Setup

1. Clone repository
2. Create virtual environment
3. Install requirements
4. Create .env file
5. Run migrations
6. Start server
